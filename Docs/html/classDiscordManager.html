Sets up the Discord thread and spawns the AutoRef worker for a specific match.

Parameters
matchIdThe unique match identifier (e.g. "A1").
refereeThe display name of the referee in charge.
guildThe Discord guild object where the thread will be created.
typeThe type of match (Qualifiers vs Elimination) which determines the logic used.
Returns
True if the environment was created; false if the match is already active.
